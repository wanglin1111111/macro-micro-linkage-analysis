```
┌─────────────────────────────────────────────────────┐
│           三层联动分析模型                            │
├─────────────────────────────────────────────────────┤
│  第一层：宏观驱动（Macro Drivers）                    │
│  ├─ 地缘政治：当事方博弈 → 传导路径 → 预期差          │
│  ├─ 货币政策：政策框架 → 统计方法 → 市场含义          │
│  └─ 流动性：成因分解 → 传导链 → 定价状态              │

│  第二层：中观传导（Meso Transmission）                │
│  ├─ 板块分化：强势板块 vs 弱势板块 → 确定性排序       │
│  ├─ 拥挤度：踩踏条件（估值透支+高杠杆+流动性收紧）     │
│  └─ 预期差：市场预期 vs 行业预期 → 操作方向           │
├─────────────────────────────────────────────────────┤
│  第三层：微观验证（Micro Validation）                 │
│  ├─ 时间维度：短期(事件) → 中期(趋势) → 长期(产业)    │
│  ├─ 验证节点：数据发布 → 政策声明 → 财报验证          │
│  └─ 风险提示：触发条件 → 应对措施                     │
└─────────────────────────────────────────────────────┘
```

## 关键方法论（一句话记住）

### 1. 多维度交叉验证
> 不信单一信息源，对比市场vs行业vs政策三方预期

### 2. 结构性思维
> 不是普涨/普跌，而是分化（强势vs弱势板块分类）

### 3. 时间维度分层
> 短期看事件，中期看趋势，长期看产业

### 4. 传导链完整性
> 宏观→中观→微观必须完整，断裂点就是风险点

## 预期差识别三步法

```
Step 1: 收集三方预期
├─ 市场预期（价格、舆论、分析师）
├─ 行业预期（高管、从业者、供应链）
├─ 政策意图（官方声明、决策逻辑）

Step 2: 对比差异
├─ 市场vs行业：谁是内行？
├─ 市场vs政策：谁是权威？
├─ 行业vs政策：谁是实情？

Step 3: 判断方向
├─ 市场过度乐观 → 做空/观望
├─ 市场过度悲观 → 做多
├─ 预期一致 → 缺乏机会
```

## 拥挤度踩踏三条件

```
踩踏触发 = 估值透支 AND 高杠杆 AND 流动性收紧

评估清单:
□ 估值透支: PE远超历史均值？
□ 高杠杆: 融资余额占比过高？
□ 流动性收紧: 美联储缩表/加息？

满足条件数:
  0/3 → 可持有
  1/3 → 警惕观察
  2/3 → 减仓避险
  3/3 → 立即清仓
```

## 时间维度策略速查

| 时间维度 | 关注点 | 特点 | 策略 |
|---------|-------|------|------|
| 短期(1-3月) | 事件、声明、数据 | 波动大、噪音多 | 事件驱动、快进快出 |
| 中期(3-12月) | 业绩、政策落地 | 趋势形成、预期修正 | 趋势跟随、仓位调整 |
| 镈期(1-3年) | 产业周期、制度变革 | 结构性变化 | 产业配置、核心资产 |

## 常见预期差类型速查

| 预期差类型 | 市场预期 | 现实/行业预期 | 操作方向 | 案例 |
|-----------|---------|--------------|---------|------|
