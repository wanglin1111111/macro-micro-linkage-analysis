# SKILL.md - 宏观中观微观三层联动分析技能
## 技能定位

**核心价值**：突破单一维度分析局限，建立宏观政策→行业板块→个股标的的完整传导链，识别预期差和结构性机会。

**适用场景**：
- 地缘政治事件对市场的影响评估
- 美联储政策变动的前瞻分析
- 板块轮动与风格切换判断
- 宏查研报/直播内容的逻辑验证
- 投资策略的多维度交叉验证

## 三层联动框架

### 第一层：宏观驱动（Macro Drivers）

#### 1.1 地缘政治分析模板
```yaml
事件:
  名称: "霍尔木兹海峡局势"
  当事方:
    - 美国: 长期封锁施压 → 谈判意愿
    - 伊朗: 强硬派 vs 革命卫队 → 内部分歧
  博弈均衡: 双方均有谈判意愿但存在条件分歧
  传导路径:
    - 短期: 运输受阻 → 供给收缩 → 价格波动
    - 长期: 谈判结果 → 产能释放 → 价格回归
  预期差:
    市场预期: 短期解决
    行业预期: 长期干扰持续
    结论: 行业预期更悲观 → 低估风险 → 做多波动率
```

#### 1.2 货币政策分析模板
```yaml
政策:
  名称: "美联储新任主席政策主张"
  理论框架:
    传统: 打压需求 → 降通胀 → 代价是衰退
    新视角: AI生产力 → 自然通缩 → 降息空间
  统计方法:
    传统: CPI整体指数
    新方法: 修剪均值/中位数 → 减少极端值干扰
  市场含义:
    干预频率降低 → 波动减少但分化加剧
    不再充当"定海神针" → 资产定价更市场化
  验证节点:
    - 新主席上位时间
    - 首次政策声明措辞
    - 通胀指标选择
    - 市场反应模式变化
```

#### 1.3 流动性分析模板
```yaml
流动性:
  现状: "低利率环境"
  成因分解:
    - 银行: 存款多贷款少 → 资金淤积
    - 监管: 同业存单利率约束 → 收益率下行
    - 银行行为: 自购存单 → 进一步压低利率
  对股市影响:
    传导路径: 低利率 → 估值支撑 → 股价上涨
    定价状态: 已在年初行情充分定价
    未来情景:
      - 情景A: 利率低位震荡 → 缺乏主线 → 赚钱难
      - 情景B: 利率上行 → 风格切换 → 科技转消费
  风险点:
    - 社融回暖 → 抽水效应
    - 政府债发行 → 流动性收缩
```

### 第二层：中观传导（Meso Transmission）

#### 2.1 板块分化分析模板
```yaml
板块分析:
  方法: "结构性分化而非普涨"
  
  强势板块:
    AI产业链:
      确定性: 最高
      盈利增速: 高 + 超预期概率提升
      拥挤度: 高但未到踩踏时刻
      策略: 持有行业ETF而非追高个股
    
    能源:
      业绩: 二三季度上升确定性高
      驱动: 地缘溢价 + 供需缺口
      风险: 暴利税政治压力
    
    金融/通信/公用事业:
      趋势: 温和上行
      特点: 被科技光芒掩盖 → 配置机会
  
  弱势板块:
    工业:
      压力: 通胀 + 高利率 → 成本上升
      预期: 下修
    
    医疗:
      预期: 下修
      原因: 政策控费 + 创新药周期
    
    消费(可选/必选):
      压力: 收入预期不稳 → 消费意愿下降
    
    房地产:
      压力: 利率敏感 + 库存高企
```

#### 2.2 拥挤度评估框架
```yaml
拥挤度判断:
  当前状态: 科技股拥挤度高
  踩踏条件(三条件齐备):
    1. 估值透支: PE远超历史均值
    2. 高杠杆: 融资余额占比过高
    3. 流动性收紧: 美联储缩表/加息
  
  当前评估:
    估值透支: 部分满足(头部公司)
    高杠杆: 未满足
    流动性收紧: 未满足
    结论: 未到踩踏时刻 → 可持有但需警惕
  
  后续演化:
    从普涨 → 内部分化
    头部公司优势强化 → 持有行业ETF
```

### 第三层：微观验证（Micro Validation）

#### 3.1 预期差博弈模板
```yaml
预期差:
  定义: "市场预期 vs 现实/行业预期"
  
  案例1-油价:
    市场预期: 海峡短期恢复通航
    行业预期: 长期干扰持续
    预期差: 市场过度乐观
    操作: 做多能源股/原油期货
  
  案例2-美联储:
    市场预期: 传统通胀目标制
    新主席主张: AI通缩论 + 修剪均值
    预期差: 政策框架可能转变
    操作: 关注首次政策声明措辞
  
  案例3-利率:
    市场预期: 利率继续下行
    现实: 已接近政策利率地板
    预期差: 下行空间有限
    操作: 准备风格切换预案
```

#### 3.2 时间维度分析
```yaml
时间框架:
  短期(1-3月):
    关注: 地缘事件、政策声明、数据发布
    特点: 波动大、噪音多
    策略: 事件驱动、快进快出
  
  中期(3-12月):
    关注: 业绩兑现、政策落地、趋势确认
    特点: 趋势形成、预期修正
    策略: 趋势跟随、仓位调整
  
  长期(1-3年):
    关注: 产业周期、制度变革、技术革命
    特点: 结构性变化、新均衡形成
    策略: 产业配置、核心资产持有

案例-油价:
  短期: 换月价格回落 → 波动
  长期: 看好能源公司 + 原油价格 → 趋势
  操作: 短期观望、长期布局
```

## 使用示例

### 示例1：地缘政治事件分析

**输入**：霍尔木兹海峡局势紧张

**分析流程**：
1. 宏查当事方博弈（美国vs伊朗，伊朗内部派系）
2. 识别传导路径（运输→供给→价格）
3. 对比市场预期vs行业预期（短期vs长期）
4. 评估板块影响（能源利好、消费利空）
5. 给出时间维度策略（短期波动、长期布局）

**输出**：
```yaml
结论:
  预期差: 市场低估长期干扰风险
  操作: 做多能源板块(二季度业绩确定性高)
  时间: 短期波动、长期持有
  风险: 暴利税政治压力、谈判超预期
```

### 示例2：美联储政策前瞻

**输入**：美联储新任主席政策主张

**分析流程**：
1. 对比传统框架vs新视角（需求打压vs AI通缩）
2. 分析统计方法变化（CPI整体vs修剪均值）
3. 推导市场含义（干预减少、分化加剧）
4. 识别验证节点（首次声明、指标选择）

**输出**：
```yaml
结论:
  政策框架转变概率: 中高
  市场含义: 波动减少但分化加剧
  操作: 关注首次政策声明措辞
  验证: 新主席上位时间、通胀指标选择
```

## 核心方法论

### 1. 多维度交叉验证
- 不依赖单一信息源
- 对比市场预期vs行业预期vs政策意图
- 识别预期差方向

### 2. 结构性思维
- 非普涨/普跌，而是分化
- 强势板块vs弱势板块分类
- 拥挤度评估避免追高

### 3. 时间维度分层
- 短期：事件驱动、波动大
- 中期：趋势跟随、预期修正
- 长期：产业配置、核心资产

### 4. 传导链完整性
- 宏观→中观→微观完整链条
- 每一环节验证传导有效性
- 识别断裂点/加速点
